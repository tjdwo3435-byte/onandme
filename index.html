Page 01
한 줄로 정리하면 이렇습니다

복잡하지 않습니다. 구조는 단순합니다.

구분 내용
진성재 정체 별도 대행사로 운영. 진로 강사 등 다양한 활동을 병행하는 독립 사업체
온앤미와의 관계 직원 아님. 온앤미 사무실 한 자리를 사용하며 함께 일하는 협업 구조
협업 방식 ① 온앤미가 원하는 IP 기획을 공동으로 진행. 실비 외 인건비는 청구 안 함 공동기획
협업 방식 ② 온앤미에 급한 기획 건이 생겼을 때 기획비·PM비를 받고 대행 처리 기획 대행
전제 온앤미가 대행을 줄 의무 없음. 진성재도 스케줄상 불가하면 수락 안 함
이탈 조건 온앤미 직원 불만 발생 또는 온앤미 성장에 문제가 된다고 판단되면 즉시 빠짐
온앤미 사무실을 공간으로 사용하지만 온앤미 소속이 아닙니다.
온앤미의 성장에 도움이 되는 선에서만 이 구조가 유지됩니다.
Page 02
IP 기획 협업

온앤미가 하고 싶은 IP 기획을 시작으로 함께 실험합니다.

항목 내용
시작점 온앤미가 원하는 IP 기획 아이템부터 시작
역할 기획·설계·실험은 진성재가 주도. 온앤미는 방향 결정 및 자원 활용
비용 실제 집행 비용만 온앤미 부담. 진성재 인건비는 청구하지 않음
레퍼런스 완성된 IP는 온앤미·진성재 공동 레퍼런스로 활용
확장 지자체·기업·학교 제안으로 이어질 경우 수익 배분 구조는 사전 협의
로컬 연계 현재 개발 중인 로컬 관광 플랫폼 누비오 활용. 5월 서비스 완성 · 6월 프로그램 업로드 예정. 완성 후 온앤미 지역축제 제안서에 바로 연계 가능
온앤미는 인건비 없이 IP 기획을 실험할 수 있습니다.
진성재는 이 과정에서 실제 IP 레퍼런스를 쌓습니다. 양쪽 모두 실이 있는 구조입니다.
Page 03
기획 대행

온앤미에 급한 기획 건이 생겼을 때, 기획비·PM비를 받고 처리합니다.

항목 내용
대상 업무 제안서·기획안·운영안·큐시트 작성 / 현장 운영 지원
비용 기획비·PM비 등 건별 사전 협의 후 진행
강점 온앤미 구조를 이미 알고 있어 별도 브리핑 없이 바로 처리 가능
온앤미 의무 없음. 필요할 때 제안하면 됨
진성재 의무 없음. 스케줄상 불가하면 수락 안 함. 수락한 건은 마감까지 책임
필수가 아닙니다. 온앤미가 필요할 때 제안하고, 가능하면 진행하는 구조입니다.
단, 수락한 일은 중간에 던지지 않습니다.

  • 고정 부담 없이 유연하게 활용

  • IP 기획 레퍼런스 축적
  • 사무실 공간 활용
  • 사업 노하우를 현장에서 배움
  • 자율적 스케줄로 독립 활동 병행
마무리
구조는 단순합니다.
진성재는 별도 대행사로 움직이고, 온앤미 사무실 한 자리를 쓰며 IP 기획을 함께 합니다.
급한 실무가 생기면 대행으로 처리하고, 맡은 일은 끝까지 책임집니다.

온앤미 성장에 방해가 되는 순간 바로 빠지겠습니다.
